# API REST With Node, Express, Sequelize and MariaDB

Run Project ->
```bash
    > npm i // install dependecies
    > npm run dev // dev environment
    > npm start // production environment
```

I used a `.env` file for the configuration with the next props:

```bash
DIALECTDB=[insert]
DATABASEDB=[insert]
USERNAMEDB=[insert]
PASSWORDDB=[insert]
HOST=[insert]
DEV_HOST=[insert]
```
